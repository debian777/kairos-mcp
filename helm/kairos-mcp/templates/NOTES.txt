{{- end -}}

Read the full secret:

Or individual keys:

  kubectl -n {{ .Release.Namespace }} get secret {{ $secretName }} -o jsonpath='{.data.session-secret}' | base64 -d; echo

  kubectl -n {{ .Release.Namespace }} get secret {{ $secretName }} -o jsonpath='{.data.keycloak-db-password}' | base64 -d; echo

  # Redis / Valkey password
