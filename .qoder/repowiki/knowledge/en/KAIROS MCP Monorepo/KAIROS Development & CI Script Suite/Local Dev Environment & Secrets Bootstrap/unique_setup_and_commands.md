`./scripts/env/create-env.sh` generates secrets-only `.env` from the template (idempotent — skips if `.env` exists). `ENV=dev|dev_simple|dev_stdio|prod ./scripts/deploy-run-env.sh [build|start|stop|restart|status|test|logs|health]` is the primary interface; `npm run dev:start` / `npm run dev:test` wrap it. `./scripts/deploy-dev-cli-ready.sh` is the end-to-end CLI readiness check. `node -r dotenv/config scripts/deploy-raw-qdrant-search.mjs "query"` runs a raw Qdrant vector search.
