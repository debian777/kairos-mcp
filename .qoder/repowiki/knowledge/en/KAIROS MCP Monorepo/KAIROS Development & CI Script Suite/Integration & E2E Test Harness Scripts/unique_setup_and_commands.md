`scripts/test-helm.sh` requires `chart-testing`, `kubeconform`, and `helm plugin install https://github.com/helm-unittest/helm-unittest --version v1.0.3` before running. `scripts/test-ai-mcp-integration.mjs` expects a live KAIROS dev server (default `http://localhost:3300`) and, when `AUTH_ENABLED=true`, either a `.test-auth-env.dev.json` written by Jest's globalSetup or `KAIROS_INTEGRATION_BEARER=<raw JWT>` in the environment. `scripts/test-embedding-key.mjs` needs `OPENAI_API_KEY` pointing at an OpenAI key with Embeddings permission only.
