`validate-mermaid.sh` requires `node_modules/.bin/mmdc` installed via `npm install @mermaid-js/mermaid-cli`. `sync-wiki.sh` needs `gh` authenticated and a pre-initialized wiki (visit `/wiki` once). `seed-test-snapshot.sh` expects a running KAIROS app and Qdrant (started by `deploy-run-env.sh`) and reads env from `.env.<ENV>` / `.env`; `import-test-snapshot.sh` is invoked by `deploy-run-env.sh` when `CI=true`.
