Pure configuration bundle — no source code — consumed by Cursor's plugin loader. `.cursor-plugin/plugin.json` is the Cursor plugin manifest declaring name, version, author, license, a `logo` reference to `assets/logo.svg`, and an `mcpServers` pointer to `mcp/mcp.json`. That second file defines one `KAIROS-DEVELOPMENT` MCP server of type `streamable-http` pointing at `http://localhost:3300/mcp`, with an `alwaysAllow` whitelist (`activate`, `forward`, `train`, `reward`, `tune`, `delete`, `export`, `spaces`) and no disabled tools. Dependency direction is outward only: this bundle points at a separately running KAIROS HTTP server; it has no runtime logic of its own.
