Workflow eval suite requires a running dev server and must be invoked with `NODE_OPTIONS='--experimental-vm-modules' npx jest --runInBand tests/integration/reward-evals.test.ts tests/integration/reward-export-evals.test.ts`. MIME export dump output is written under `<cwd>/.local/mime-fixture-export/<pid>/<transport>/<format>/stage<N>/` and can be redirected via the `KAIROS_MIME_FIXTURE_EXPORT_DIR` env var; per-worker cleanup is triggered through `ensureMimeFixtureExportDumpRootCleanBeforeMimeTests`.
