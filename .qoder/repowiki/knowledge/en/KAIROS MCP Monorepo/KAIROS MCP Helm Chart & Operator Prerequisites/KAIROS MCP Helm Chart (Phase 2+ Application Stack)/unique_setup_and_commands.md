For ArgoCD users importing the Keycloak realm, `spec.source.helm.enableLookup: true` must be set on the HelmRelease — without it `lookup()` returns empty and the realm CR re-renders every sync, defeating the first-install-only guard. The chart also ships a pre-install `operator-precheck-job` (disabled by default via `hooks.operatorPrecheckEnabled=false`) that can be enabled to validate required operators before rendering.
