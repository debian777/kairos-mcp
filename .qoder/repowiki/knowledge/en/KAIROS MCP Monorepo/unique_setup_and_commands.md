`npm run infra:up` boots the Compose stack and runs `scripts/deploy-configure-keycloak-realms.py` to create realms idempotently. `npm run dev:*` / `dev_simple:*` / `dev_stdio:*` all delegate to `scripts/deploy-run-env.sh <action>` which sets `ENV=<profile>`, loads the matching `.env.*`, then calls start/restart/logs/status/test against the running stack. `npm run handoff` is the CI-like gate: deploy → test → ensure-coding-rules → knip → markdown lint → mermaid validation. Version bumps go through `npm run release:<major|minor|patch|rc|beta>` which also runs `version:sync` to update skills refs, compose image tag, and Helm app version.
